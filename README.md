# PrajaVaradhi - Citizen Politician Bridge

This project is organized into two main components:

## 1. Backend (Node.js)
The core API server built with Express and MongoDB.
- **Location:** `/backend`
- **Setup:** `npm install` and `npm run dev`
- **Features:** Authentication, Issue Tracking, Notifications, and Progress Updates.

## 2. Frontend
The modern web application built with HTML, CSS, and Javascript.
- **Location:** `/frontend`
- **Features:** Role-based dashboards (Citizen, Moderator, Admin, Public Representative), Issue Reporting, and Budget Tracking.

---
### Quick Start
1. Ensure MongoDB is running.
2. Initialize backend: `cd backend && npm install && npm start`
3. Open `frontend/index.html` in your browser.
